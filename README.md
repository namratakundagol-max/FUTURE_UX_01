This project is a UI/UX redesign of a local home cleaning service website with a strong focus on lead generation, conversion optimization, and clean user experience.

## Pages Included
- Homepage (Hero, Services, Why Choose Us)
- Services Page
- Contact / Lead Page

## Features
- Clear value proposition with CTA
- Service cards with icons
- Trust section
- Lead capture form
- Modern responsive UI

## Design Thinking
The design is structured to guide users from understanding the service to taking action through clear layout, spacing, and CTA placement.
